HMV-Chromee靶机渗透记录

Lark2026-03-242026-03-24

信息搜集

ip地址为<INTERNAL_IP>，接着对端口进行信息搜集

发现22，80，8080和23333端口是开放的，先看看80和8080端口的情况

接着使用 gobuster 进行目录扫描，发现了 /post.php 和 /secret.php 两个接口，其中 /secret.php 会返回一段 PHP 逻辑。

获取 User Flag

访问/server的时候显示403，换了一种请求方式发现变成301，在后面加个/index.html发现访问成功，并且得到一个邮箱地址，有用的应该是这个chromee.hmv，写到hosts文件里，再去访问http://chromee.hmv/secret.php

此时发现secret.php被解析了，尝试传入参数，貌似不管输入什么参数，都是回显

这里出现2个人名，adriana和Lily，使用cupp生成2个字典，把他放到一个文件中，然后对23333端口进行爆破

这里我先猜测 adriana 可能是用户名，爆破出对应口令后登录 FTP，接着在目录里找到了两个本地用户以及一份 zeus.conf。

ftp好难用，不支持cat命令，使用lftp连上去，有一个...的文件

发现是密钥，拿到本地给他爆破一下

私钥口令已经脱敏。由于当前只拿到了两个候选用户，逐个尝试后发现 follower 可以成功登录。

这里也得到一个提示，还有一个gif图片，使用nc的方式把这个图片弹到kali中

既然是gif，那试试帧分离，发现不是gif帧分离，看一下exif信息

Comment: Imagemagick Convert Timeline发现这个，直接去搜一下看看有没有相关文章，找到一篇

https://www.cnblogs.com/chtxrt/p/17280677.html 先使用identify对图片进行分析

这里主要看每一帧的时间间隔，很容易联想到 ASCII。把数值转成字符后，再结合 note.txt 里提到的 47，基本可以判断还需要做一次 rot47。得到 follower 的口令后，就可以结合前面 zeus.conf 中 permit follower as softly cmd /usr/local/bin/wfuzz 这条规则继续提权。

找到这个file.py是可以写入的，这里是作者忘记删掉了，我们也可以重写写入，然后使用doas这个工具运行命令,doas这个工具是可以指定其他用户来执行命令，这里指定的是softly这个用户

这样我们就拿到了softly用户的shell，然后就可以拿下user flag了

提权

先拿个pspy64过来看一下有没有定时任务

可以看到 root 会定时执行 /root/script.sh。前面在 zeus.conf 里还拿到了 permit nopass :softly as root cmd /usr/bin/chromium，因此这里可以直接用无头模式的 chromium 去读 root 目录下的文件。直接读脚本本体没有结果，于是先列目录，再去查看其中的 script.js。

这里拿到了一段经过编码的 root 凭据，原始内容已经脱敏处理。
